# Project

This is just a template I use when making full stack web apps. It is React, Node, postgres, and Docker.

\
Install prettier vsc extension for formatting:

```
https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode
```

Install Node.js

# Frontend

To install dependencies

```
npm i
```

\
To run locally

```
npm run dev
```

# Backend

You must install the Docker desktop app

## local:

Run node and postgres locally

```
npm run docker
```

## prod:

Compile TypeScript to JavaScript:

```
npm run build
```

\
Turn files into zip

```
npm run zip
```

\
run js code on prod

```
npm start
```
